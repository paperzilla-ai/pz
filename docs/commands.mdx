# Commands

## pz login

Authenticate with your Paperzilla account using email OTP.

```bash
pz login
```

Prompts for your email address, sends a one-time code, and stores the session locally at `~/.paperzilla/tokens.json`. The session refreshes automatically — you should only need to log in once.

If your session expires while running any other command, the login flow starts automatically.

## pz feed \<project-id\>

Show the curated paper feed for a project.

```bash
pz feed <project-id> [flags]
```

### Flags

| Flag | Short | Type | Description |
|------|-------|------|-------------|
| `--json` | `-j` | bool | Output the full feed as JSON |
| `--must-read` | `-m` | bool | Only show must-read (class 2) papers |
| `--since` | `-s` | string | Only papers with `ready_at` on or after this ISO date |
| `--limit` | `-n` | int | Maximum number of results (API default: 20, max: 100) |
| `--atom` | | bool | Print an Atom feed URL for use in feed readers |

### Examples

Browse the feed:

```bash
pz feed a1b2c3d4-e5f6-7890-abcd-ef1234567890
```

Only must-read papers from the last week, limit to 5:

```bash
pz feed a1b2c3d4-e5f6-7890-abcd-ef1234567890 --must-read --since 2025-07-25 --limit 5
```

Export as JSON:

```bash
pz feed a1b2c3d4-e5f6-7890-abcd-ef1234567890 --json
```

Subscribe in a feed reader:

```bash
pz feed a1b2c3d4-e5f6-7890-abcd-ef1234567890 --atom
```

### Atom feed

The `--atom` flag prints a URL you can paste into any feed reader ([Vienna RSS](https://github.com/ViennaRSS/vienna-rss), NetNewsWire, Feedly, etc.):

```
https://paperzilla.ai/api/feed/atom/a1b2c3d4-e5f6-7890-abcd-ef1234567890?token=pzft_...
```

The URL contains a personal feed token so the reader can poll it without needing to log in. The token is per-user (not per-project), so you can use it across all your projects. Running `--atom` multiple times returns the same token.

The feed includes up to 50 recent papers, with titles, authors, links, and relevance labels. Feed readers that support ETag caching will get `304 Not Modified` responses when nothing has changed.

### Default output

```
Machine Learning Papers — 12 papers (total: 142)

★ Must Read  A Novel Approach to Transformer Efficiency
  Smith et al. · arxiv · 2025-08-01 · relevance: 92%

○ Related  On the Convergence Properties of Diffusion Models
  Chen et al. · arxiv · 2025-07-30 · relevance: 74%
```

Each paper shows:
- **Relevance class** — `★ Must Read` or `○ Related`
- **Title** — truncated to 80 characters
- **First author** — surname with "et al." if there are co-authors
- **Source** — arxiv, biorxiv, medrxiv, or chinaxiv
- **Published date**
- **Relevance score** — percentage match to your interests
